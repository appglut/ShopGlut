== Installation ==

**Automatic Installation (Recommended):**
1. Go to your WordPress admin ‚Üí Plugins ‚Üí Add New
2. Search for "ShopGlut Builder and Customized Solutions"
3. Click "Install Now" and then "Activate"
4. Navigate to ShopGlut ‚Üí Single Product to start designing product pages
5. Access all six modules from the ShopGlut menu: Single Product, Wishlist, Product Custom Fields, Product Swatches, Product Badges, and Product Comparison

**Manual Installation:**
1. Download the plugin zip file from WordPress.org
2. Upload via WordPress admin ‚Üí Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Or extract to `/wp-content/plugins/shopglut/` directory
4. Activate through the Plugins menu in WordPress
5. Access all six modules through the ShopGlut admin menu

**Quick Start:**
1. Go to ShopGlut in admin menu
2. Choose a module (Single Product, Wishlist, Product Custom Fields, Product Swatches, Product Badges, or Product Comparison)
3. Create or edit a layout
4. Customize using the visual builder
5. Apply changes and preview

For detailed setup instructions, visit [documentation](https://documentation.appglut.com/shopglut/).

**Requirements:**
* WordPress 5.1+ (WordPress 6.0+ recommended)
* WooCommerce 3.0+ (latest version recommended)
* PHP 7.4+ (PHP 8.0+ recommended for better performance)
* MySQL 5.6+ or MariaDB 10.0+
* Memory limit: 128MB minimum (256MB+ recommended)
* Modern web browser with JavaScript enabled

== Frequently Asked Questions ==

= Is coding knowledge required to use this plugin? =
No! Both the single product page builder and wishlist system are designed for users of all skill levels. The intuitive interface makes it easy to create professional product pages and configure advanced wishlist settings without any coding knowledge.

= Will this plugin slow down my website? =
No. ShopGlut is performance-optimized with conditional loading, efficient database queries, and modern caching techniques. The plugin only loads necessary scripts when needed, and most users actually see improved performance due to better code optimization compared to theme defaults.

= Does the page builder work with all WooCommerce product types? =
Absolutely! The plugin fully supports all WooCommerce product types including simple products, variable products (with attributes), grouped products, and external/affiliate products. Each product type has specialized layout options and form handling.

= How do I display the wishlist on my website? =
Create a new page in WordPress and add the shortcode `[shopglut_wishlist]` where you want the wishlist to appear. You can also set this page as the default wishlist page in the plugin settings under ShopGlut ‚Üí Wishlist Options.

= Can guests use the wishlist without creating an account? =
Yes! The plugin supports guest users through secure cookie-based storage that lasts for 30 days. When a guest user registers or logs in, their wishlist items are automatically merged with their account, ensuring no saved products are lost.

= Is the plugin mobile-responsive? =
Yes! All layouts, wishlist features, and admin interfaces are mobile-first and automatically optimize for tablets and smartphones. You can also customize mobile-specific settings and preview your changes on different device sizes.

= Can I customize the appearance of wishlist buttons? =
Absolutely! Go to ShopGlut ‚Üí Wishlist Options ‚Üí Appearance to customize button colors, icons, text, positioning, hover effects, and styling. You can choose from text-only, icon-only, or combined text+icon button styles.

= Do customers receive notifications about their wishlist items? =
Yes! Registered users can subscribe to various email notifications including stock availability alerts, price drop notifications, product updates, and low stock warnings. Users can manage their notification preferences, and all emails comply with anti-spam regulations.

= Can I customize colors, fonts, and layouts? =
Yes! The plugin includes comprehensive customization options for colors, typography, spacing, borders, and layouts. The Pro version adds advanced styling options including gradient backgrounds, Google Fonts integration, and custom CSS/JavaScript injection.

= Does it work with my existing theme? =
ShopGlut is designed to work seamlessly with any well-coded WordPress theme. The plugin integrates without conflicting with your theme's design and includes theme compatibility checks. If you experience any styling issues, our extensive customization options can resolve them.

= Is the wishlist table customizable? =
Yes! You can show/hide specific columns (product image, name, price, quantity, availability, discount info, reviews, etc.), reorder columns, customize table colors and fonts, and set up sorting and filtering options.

= Can I preview changes before publishing? =
Yes! The single product page builder includes live preview functionality so you can see exactly how your product pages will look on desktop, tablet, and mobile before making them live. The wishlist system also includes preview options for button styling.

= How do I enable social sharing for wishlists? =
Social sharing is enabled by default and includes Facebook, Twitter(X), WhatsApp, Pinterest, LinkedIn, Telegram, and Email sharing options. Customers can share their wishlist with friends and family without any additional configuration required.

= Can I restrict wishlist access to logged-in users only? =
Yes, you can enable the "Require Login" option in ShopGlut ‚Üí Wishlist Options ‚Üí General Settings to restrict wishlist functionality to registered users only. You can also customize the login requirement message and button.

= Do you provide customer support? =
Yes! We offer free support through the WordPress.org support forums for all users. Pro users receive priority support through our dedicated support portal with faster response times and advanced troubleshooting assistance.

= Can I use this plugin on multiple websites? =
The free version can be used on unlimited websites. The Pro version offers different licensing options including single site, 5-site, and unlimited site licenses for agencies and developers.

= How do I set up product swatches for my variations? =
Navigate to ShopGlut ‚Üí Product Swatches to enable and configure swatches globally. For specific attributes, go to Products ‚Üí Attributes, edit your attribute, and choose the swatch type (color, image, label, or bicolor). Then edit individual attribute terms to set their colors, images, or labels. You can also override settings for individual products using the "Swatches" tab in the product editor.

= Can I display swatches on shop and category pages? =
Yes! Enable "Swatch Shop/Archive Page" in the Product Swatches settings. You can customize the position (before/after title or price), alignment, and choose which attributes to display. Customers can select variations directly from shop pages without visiting individual product pages.

= What swatch types are supported? =
ShopGlut supports four swatch types: Color swatches (solid colors), Image swatches (thumbnail images), Bicolor swatches (dual-color gradients), and Label swatches (text-based). You can mix different swatch types for different attributes on the same product.

= Can I customize the appearance of swatches? =
Absolutely! Control swatch dimensions, shapes (circle, square, rounded, custom), border styles, colors, and hover effects. Configure tooltips, typography, spacing, and positioning. Each attribute can have its own custom settings, and you can override global settings for individual products.

= Do swatches work with out-of-stock variations? =
Yes, ShopGlut intelligently handles stock status. You can choose to hide out-of-stock swatches completely or show them with disabled effects like blur, cross-out, or opacity reduction to indicate unavailability while maintaining the visual layout.

= How do I enable product comparison on my site? =
Navigate to ShopGlut ‚Üí Layouts ‚Üí Comparison in your admin dashboard. Create a new comparison layout and customize the comparison table styling, fields, and display options. You can set different layouts for shop pages, specific categories, or product tags. Enable the floating comparison bar to give customers quick access to their selected products.

= Can I customize which product information appears in the comparison table? =
Absolutely! You can choose which fields to display in the comparison table including product image, title, price, rating, stock status, SKU, dimensions, weight, description, and custom attributes. Simply enable or disable fields in the comparison layout settings to control what information is shown.

= How many products can customers compare at once? =
By default, customers can compare up to 4 products simultaneously for optimal viewing. The comparison table is fully responsive and automatically adjusts to different screen sizes to ensure a great experience on mobile, tablet, and desktop devices.

= Can I have different comparison layouts for different pages? =
Yes! You can create multiple comparison layouts and assign them to specific locations like the main shop page, individual product categories, or product tags. Only one layout can be active per location, and the system prevents conflicts by showing which locations are already taken by other layouts.

= How do I add product badges to my products? =
Navigate to ShopGlut ‚Üí Product Badges to enable and configure custom product labels. Create badges for sales, new products, featured items, out-of-stock status, and more. You can customize badge colors, shapes, positions, and animations. Badges can be applied globally to all products or individually to specific products based on categories, tags, or custom conditions.

= Can I create custom fields for my products? =
Yes! Go to ShopGlut ‚Üí Product Custom Fields to add custom fields to your products. Choose from various field types including text, textarea, number, date, select, checkbox, and more. Custom fields can be displayed on product pages, shop archives, and in product data. You can also use these fields in product variations and export them with product data.

= How do I use the Shortcode Showcase module? =
Navigate to ShopGlut ‚Üí Shortcode Showcase in your admin dashboard. Here you can find comprehensive documentation for all available shortcodes including [shopglut_woo_category] for displaying products from specific categories. The module provides live preview functionality, detailed parameter explanations, and copy-to-clipboard features for easy implementation.

= What shortcodes are available in the Shortcode Showcase module? =
The primary shortcode is [shopglut_woo_category] which allows you to display products from specific WooCommerce categories. You can customize columns, pagination, sorting, filtering, and use custom Woo Templates for unique product displays. Additional shortcodes include [shopglut_archive] for latest products and [shopglut_product_table] for tabular product displays.

= Can I customize the appearance of shortcode output? =
Absolutely! The Shortcode Showcase module integrates seamlessly with Woo Templates, allowing you to create custom product display templates. You can control grid layouts, colors, typography, spacing, and all visual aspects. The module also includes built-in toolbar options for search, filtering, and sorting with customizable styling.

= Do shortcodes work with page builders like Elementor or Gutenberg? =
Yes! All ShopGlut shortcodes are compatible with major page builders and the WordPress block editor. You can add shortcodes to any page, post, or widget area. The module provides a user-friendly interface for copying shortcodes and includes live preview functionality to see exactly how they will appear on your site.
